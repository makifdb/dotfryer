## Dotfryer

Dotfryer is a dotfile manager for your dotfiles

### Planned arguments

Current Status

```bash 
$ dotfryer
```

Install

```bash
$ dotfryer install -f dotfiles.yaml -l .dotfiles
$ dotfryer install -c github.com/makifdb/dotfiles/dotfiles.yaml
```

Update config files

```bash
$ dotfryer synce ||  dotfryer -u
```

Clean config files on system

```bash
$ dotfryer clean || dotfryer -c
```
